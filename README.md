## Overview

PrepFlow helps kitchen managers build accurate, efficient prep lists for every shift using AI. It factors in covers, menu mix, historical usage, and current inventory to generate a prioritized prep plan — eliminating over-prepping, under-prepping, and guesswork.

## Planned Features

- 🤖 **AI Prep Generation** — Claude-powered prep lists based on shift forecast
- 📅 **Shift Scheduling** — link prep tasks to specific shifts and staff
- 📦 **Inventory Awareness** — integrates with CulinaryOS inventory data
- ⏱️ **Time Estimates** — per-task time tracking and total shift workload
- 📈 **Historical Learning** — improves suggestions based on past shift data
